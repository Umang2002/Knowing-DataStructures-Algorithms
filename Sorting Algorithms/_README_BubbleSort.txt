
*********************BUBBLE SORT*********************

1]Simplest sorting algorithm.
2]Sorting is done by iterating 1st element to the last element & then swapping if necessary.
3]The iteration continues untill no more swaps are added.
4]But since it has highest time complexity is not as much efficient.
5]Got its name Bubble Sort since smaller elements “bubble” to the top of the list. 


6]Performance
    Worst case complexity : O(n2)
    Best case complexity (Improved version) : O(n)
    Average case complexity (Basic version) : O(n2)
    Worst case space complexity : O(1) auxiliary    

7]In the provided code ,Improved version of Bubble Sorting is Perfored with the use of extra flag to indicate the sorted list.
